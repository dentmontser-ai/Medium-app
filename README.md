# 🏥 MedSim - منصة الاختبارات الطبية الذكية

منصة ويب متقدمة لتوليد اختبارات طبية تجريبية باستخدام الذكاء الاصطناعي. تم تصميمها لمساعدة طلاب الكليات الطبية على المراجعة والتحضير للامتحانات.

## ✨ الميزات الرئيسية

- **🤖 توليد ذكي للأسئلة:** استخدام نموذج Gemini AI لإنشاء أسئلة متنوعة وذات جودة عالية
- **💾 قاعدة بيانات ذكية:** تخزين الأسئلة وإعادة استخدامها لتسريع الخدمة
- **📱 واجهة سهلة الاستخدام:** تصميم عصري ومتجاوب يعمل على جميع الأجهزة
- **⚡ أداء سريع:** استضافة على Vercel مع شبكة توصيل محتوى عالمية
- **🎯 تخصصات متعددة:** دعم طب بشري، أسنان، صيدلة، مختبرات، تمريض، وغيرها
- **📊 نتائج فورية:** تصحيح فوري مع عرض الإجابات الصحيحة والخاطئة

## 🛠️ التقنيات المستخدمة

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI:** Google Gemini API
- **Hosting:** Vercel

## 📋 المتطلبات

- Node.js 18.x أو أحدث
- حساب Google Cloud مع API Gemini مفعل
- حساب MongoDB Atlas
- حساب Vercel (اختياري للنشر)

## 🚀 البدء السريع

### 1. استنساخ المستودع

```bash
git clone https://github.com/dentmontser-ai/Medium-app.git
cd Medium-app
```

### 2. تثبيت المكتبات

```bash
npm install
```

### 3. إعداد متغيرات البيئة

أنشئ ملف `.env` في الجذر:

```env
GOOGLE_API_KEY=your_gemini_api_key_here
MONGO_URI=your_mongodb_connection_string_here
```

### 4. تشغيل الخادم محلياً

```bash
npm start
```

افتح المتصفح واذهب إلى: `http://localhost:3000`

## 🔧 كيفية الاستخدام

1. **اختر التخصص:** حدد تخصصك الطبي من القائمة المنسدلة
2. **اختر السنة الدراسية:** حدد سنتك الحالية
3. **أدخل اسم المادة:** اكتب اسم المادة التي تريد الاختبار فيها
4. **اضغط "أنشئ نموذجي":** سيتم توليد 20 سؤالاً تلقائياً
5. **أجب على الأسئلة:** اختر الإجابات الصحيحة
6. **اعرض النتائج:** احصل على نتيجتك والإجابات الصحيحة

## 🌐 النشر على Vercel

### الخطوة 1: ربط GitHub

```bash
git push origin main
```

### الخطوة 2: إنشاء حساب Vercel

اذهب إلى [vercel.com](https://vercel.com) وسجل دخولك باستخدام GitHub

### الخطوة 3: استيراد المشروع

- اضغط "Add New" ثم "Project"
- اختر مستودع `Medium-app`
- أضف متغيرات البيئة:
  - `GOOGLE_API_KEY`: مفتاح Gemini API
  - `MONGO_URI`: رابط اتصال MongoDB

### الخطوة 4: النشر

اضغط "Deploy" وانتظر اكتمال النشر
